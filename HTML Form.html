HTML Form

HTML Form adalah elemen dalam HTML yang digunakan untuk mengumpulkan data dari pengguna dan mengirimkannya ke server untuk diproses. Elemen ini biasanya digunakan pada halaman web untuk membuat fitur interaktif, seperti kotak pencarian, login, pendaftaran, atau pengumpulan data lainnya.

Elemen <form>

Elemen utama yang digunakan untuk membuat formulir.

Atribut penting: action (URL untuk mengirim data formulir), method (metode pengiriman data: GET atau POST).

action adalah URL untuk mengirim data formulir dan menentukan aksi yang akan dilakukan saat data dikirim

method metode pengiriman data.

<form action="proses.php" method="GET">      </form>

Untuk atribut action, kita dapat mengisinya dengan alaman URL dari endpoint yang akan memproses form.Secara sederhana,—pada contoh di atas— kita akan menyuruh file proses.php untuk memproses data form.Ini nanti akan kita pelajari pada PHP.

Field

Field adalah ruas yang dapat diisi dengan data.

Contoh field:

<input type="text" name="info">

Field memiliki beberapa atribut yang harus diberikan:

  1. type merupakan type dari field.
  2. name merupakan nama dari field yang akan menjadi kunci dan variabel di dalam program.

    Hasil pada browser:

    Pada kode di atas, kita mempelajari tiga buah filed baru:

    1. Field Radio

      2. Input bertipe radio digunakan untuk memilih satu opsi dari beberapa pilihan. Pada form, radio button didefinisikan dengan tag <input type="radio">, di mana setiap opsi memiliki nilai yang berbeda, dan hanya satu yang dapat dipilih dalam satu waktu.

    2. Field <select><option>

      3. Kombinasi tag <select> dan <option> digunakan untuk membuat dropdown (menu pilihan). Tag <select> mendefinisikan menu, dan tag <option> menentukan opsi yang tersedia. Pengguna dapat memilih salah satu dari opsi yang ditampilkan.

    3. Field <textearea>

      4. Tag <textarea> digunakan untuk membuat area teks yang lebih besar, memungkinkan pengguna untuk memasukkan beberapa baris teks. Ini berbeda dari input tipe text biasa yang hanya menerima satu baris teks.

    Apa bedanya radio dengan checkbox?

    Jika kita ingin agar pengunjung memilih salah satu, maka kita gunakan radio. Tapi kalau kita ingin pengunjung memilih lebih dari satu, maka kita gunakan checkbox. Lalu untuk <select></option>, sifatnya sama seperti radio. Cuma dia bentuknya berbeda. Lalu untuk menginputkan teks yang panjang, kita gunakan tag <textarea>.

    Hasil pada browser:

    Pada kode di atas, kita mempeljari tiga buah filed baru:

    1. Field URL

      2. Field URL menggunakan <input type="url">, yang memungkinkan pengguna memasukkan sebuah alamat URL (Uniform Resource Locator). Browser secara otomatis akan melakukan validasi untuk memastikan bahwa data yang dimasukkan mengikuti format URL yang benar (misalnya, harus mengandung "http://" atau "https://").

    2. Field Date

      3. Field Date menggunakan <input type="date">, yang memungkinkan pengguna untuk memilih tanggal melalui kalender pop-up yang muncul di browser. Format tanggal yang dimasukkan otomatis sesuai dengan standar (YYYY-MM-DD), dan pengguna tidak perlu mengetiknya secara manual.

    3. Field Number

      4. Field Number menggunakan <input type="number">, yang memungkinkan pengguna untuk memasukkan angka. Field ini juga bisa dibatasi dengan atribut min dan max untuk menetapkan rentang angka yang diperbolehkan, dan dilengkapi dengan panah kecil untuk memudahkan pengguna dalam menaikkan atau menurunkan angka yang dimasukkan.
