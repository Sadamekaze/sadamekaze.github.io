Materi yang akan dipelajari

Dalam kelas belajar dasar HTML untuk pengembangan website, kita akan mempelajari materi-materi dasar seputar HTML. Materi dalam kelas ini dibagi menjadi 5 modul. Berikut adalah modul-modul yang akan dicakup dalam kelas ini.

  1. Module 1: Browser dan Text editor

    2. Pada modul ini, kita akan mempelajari berbagai hal seputar text editor dan browser, mulai dari pengertian text editor dan browser, jenis-jenis text editor yang umum digunakan untuk pemrograman, serta langkah-langkah instalasi text editor.

  2. Module 2: Pengenalan HTML

    3. Pada modul ini, kita akan mempelajari dasar-dasar HTML, seperti pengertian HTML, fungsi, dan cara penggunaannya untuk membuat struktur halaman web. Kita juga akan mengenal beberapa tag dasar yang umum digunakan dalam HTML untuk menampilkan teks, gambar, dan tautan di halaman website.

  3. Module 3: HTML Container

    4. Pada modul ini, kita akan belajar tentang penggunaan <div>, class, dan id di HTML. <div> digunakan untuk membungkus elemen, sedangkan class dan id berfungsi memberi identitas agar kita bisa mengatur gaya dan tata letak elemen dengan lebih mudah.

  4. Module 4: HTML Form

    5. Pada modul ini kita akan mempelajari cara membuat HTML Form, yang digunakan untuk mengumpulkan dan mengirimkan data pengguna. Kita akan membahas elemen-elemen form seperti input, checkbox, dan tombol submit, serta cara memproses data dari form tersebut.

  5. Module 5: HTML Multimedia

    6. Pada modul ini, kita akan mempelajari berbagai hal seputar HTML Multimedia, seperti cara menampilkan video dan audio di halaman web, menyematkan konten eksternal seperti peta dan video menggunakan iframe, serta memahami atribut dan keamanan yang berkaitan dengan penyematan multimedia di dalam HTML.
